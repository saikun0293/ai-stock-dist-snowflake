This directory contains Jupyter notebooks for data exploration, analysis, and model development.

- Initial data exploration
- Statistical analysis

### 02_consumption_analysis.ipynb (Coming Soon)

### 03_forecasting_model.ipynb (Coming Soon)

## Setup

### Install Jupyter
```bash
pip install jupyter notebook pandas numpy matplotlib seaborn plotly
```

### Launch Notebook
```bash
cd notebooks
jupyter notebook
```

## Usage

1. Start with `01_data_exploration.ipynb` to understand the data
2. Use notebooks to prototype new features
3. Export insights to documentation

## Best Practices

- Document your analysis
- Save visualizations
- Share insights with the team
- Version control notebooks (use nbstripout to remove outputs)
